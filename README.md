# Godot-Code-Snippet
一个Godot自定义代码片段的插件

- [x] 基本的自定义代码片段自动补全功能
- [x] 自定义规则匹配，将匹配的字符串添加到自动补全中
- [ ] 更好的自定义代码片段体验
- [ ] 自定义代码片段生成器

### 如何开始
1. 克隆或者下载插件
2. 打开压缩包，将addons文件夹放入你的项目根目录中
3. 启动引擎，启动插件
4. 修改 'config.json' 文件进行你的自定义代码片段配置和规则匹配
